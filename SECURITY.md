# Security Policy

## Reporting a Vulnerability

We take the security of `mq` seriously. If you have discovered a security vulnerability, please do not disclose it publicly until we have had a chance to address it.

Please report any security vulnerabilities by emailing **gonzalo.mono@gmail.com**.

We will acknowledge receipt of your report within 48 hours and will strive to provide a detailed response or a fix within 7 days.

## Supported Versions

We support the latest major version of the library. Please ensure you are using the latest release before reporting an issue.
